# 华为“五看三定”分析

## 五看

1. 行业/趋势：区块链资产安全需求上升，钱包授权风险频发。
2. 市场/客户：DeFi 用户、钱包持有者、安全研究者。
3. 竞争：Revoke.cash 等已有工具多为网页端，缺乏自动化脚本。
4. 自己：具备 Python 编程、资产安全分析、工具整合能力。
5. 机会：打造轻量级自动化工具，适合教育、研究和个人使用。

## 三定

1. 定控制点：资产授权检测与撤销自动化。
2. 定目标：完成一个可运行的 Python 工具，并在 GitHub 上发布。
3. 定策略：
   - 快速开发 MVP
   - 邀请同学参与讨论与测试
   - 发布到 GitHub 并撰写清晰文档
